1. [ ] Update the system
  * Ubuntu

## 2. Secure remote root log in

1. [ ] Create new user
  * Ubuntu
  * CentOs
2. [ ] Add new user to sudoers
  * Ubuntu
  * CentOs
3. [ ] Disable remote root log in
  * Ubuntu
  * CentOs
4. [ ] Restart SSH
  * Ubuntu
  * CentOs
5. [ ] Log out and reconnect with new user
  * Ubuntu
  * CentOs
